# InsightForge   
**AI-Powered Data Intelligence & Analysis Platform**

InsightForge is a modular, agent-driven data analysis system that automates **data ingestion, cleaning, SQL analysis, visualization, and insight generation** using AI-assisted workflows.  
It is designed to reflect **real-world data analyst / AI engineer pipelines**, not toy demos.

---

## What InsightForge Does

- Accepts raw datasets (CSV)
- Cleans and preprocesses data
- Performs SQL-style analytical reasoning
- Generates visual insights
- Uses agent-based orchestration for task separation
- Designed for extensibility and production deployment

---

## Architecture Overview

InsightForge follows an **agentic design** with clear responsibility boundaries:

User Input
↓
Graph Builder (Agent Orchestrator)
↓
| Data Analyst Agent |
| Data Engineer Agent |
| SQL Reasoning Agent |
| Visualization Agent |
↓
Final Insights & Outputs


Each agent operates independently, making the system:
- Scalable
- Testable
- Easy to extend
